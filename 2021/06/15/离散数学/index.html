书本知识的梳理。

¶集合与函数

  • 基数的概念：A和B有相同的基数，当且仅当两者之间存在一个双射函数。

  • 可数集合： 一个集合可数/与自然数集有相同的基数。例如：正有理数集

  • 不可数集： 实数集。任何含有不可数子集的集合不可数。

  • Schroder-Bernstein定理： 如果在集合 A 和 B 之间存在单射f : A → B 和 g : B → A，则存在一个双射 h : A → B。从势的角度来看, 这意味着如果 |A| ≤ |B| 并且 |B| ≤ |A|，则 |A| = |B|，即A与B等势。显然，这是在基数排序中非常有用的特征。

    为何正有理数集可数？

    • 因其与自然数同基。（n * n矩阵）

¶Q：一些疑惑：

  • 十进制表示中不包含0的实数：不可数
  • 十进制表示中，仅包含有限个1的实数：不可数
  • 全体整系数二次三元多项式构成的集合 是 可数集。（yes or no）
    • 因其构成两个可数集的笛卡尔积。书P121 例题31性质

¶关系

  • 关系的概念：集合A、B之间存在图示，则关系与图示一一对应，且箭头发出位置为关系的第一个元素。

    由此可见：关系是笛卡尔积的子集。

  • R与S的合成：R: A—>B， S: B—>C，(a, c) 的集合构成的关系记作：S ° R。

  • 自反、对称、反对称、传递。

    • 问：n元素集合上有多少自反的关系？——2 n( n - 1)个 P 215

  • 用关系矩阵表示关系：

    • 自反：对角线元素均为1
    • 对称：关于主对角线对称
    • 反对称：关于主对角线没有对称元素
    • 复合关系：转化为矩阵的乘积，S×R。

  • 用有向图表示关系：

    • 自反：每个顶点都有环
    • 对称：每两个顶点形成环
    • 反对称：每两个顶点都不形成环
    • 传递：A到B， B到C，有A到C的弧。

  • 关于__传递闭包 __：

    • 定义：R* = ∪Rn (注：Rn 指从a->b存在一条长度为N的通路)
    • 关系R的传递闭包为R*
    • 集合A上的关系R是传递的，当且仅当:
    • 设集合A有n个元素，则从a->b的路径不超过n, 若a ≠ b,则路径不超过n - 1。
